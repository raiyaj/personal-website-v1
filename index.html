Local Weather App

See the Pen Local Weather App by Raiya Jessa (@raiyaj) on CodePen.

This app uses the browser's built-in navigator to access the user's current location, which is a parameter in a jQuery .getJSON() call to the OpenWeatherMap API. With JavaScript, HTML elements — including weather conditions and sunrise/sunset in local time — are then modified based on the current weather data received from the API.

This app delivers a beautiful and simple interface, featuring a loading animation and toggle switch styled solely with CSS, as well as an owfont icon (symbol font for the OpenWeatherMap API) that is reflective of day or night based on the current time.



Random Quote Machine

See the Pen Random Quote Machine by Raiya Jessa (@raiyaj) on CodePen.

A simple app that generates random quotes pulled from an API. Each time a new quote is requested, I use the jQuery .getJSON() method to receive a unique JSON object and then modify or add HTML elements with JavaScript. Features a Twitter intent button, allowing the user to tweet the quote and its author with a single click.

Mount Baker

During summer 2013, I partcipated in an inspiring and exhilarating twelve-day science and mountaineering expedition on Mt. Baker, Washington, with the wilderness educational organization Girls on Ice. In between exploring the glacier surrounding our basecamp and learning how to self-arrest with ice axes, my team summited Sherman Peak and carried out a field science study investigating the mountain's alpine ecosystem. Later, we presented our findings at the North Cascades Environmental Learning Center in Rockport, WA.
