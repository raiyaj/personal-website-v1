how to add github stuff:

make changes in Atom and preview locally on web:
git add index.html (eg.all files w/changes. can also do git status to see which files were modified)
git commit -m "change made"
git push

open files w/atom:
atom file (eg.atom index.html)

For bullet list commit message:
git commit
i (for insert mode)
type title, enter
bullet list w/ dash space
